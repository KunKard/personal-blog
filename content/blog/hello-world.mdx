## 为什么选择独立游戏开发

独立游戏开发给了我最大的创作自由。从美术到程序，从音效到关卡设计，每一个环节都可以按照自己的想法来实现。

## 我的工具链

目前我主要使用的工具：

- **游戏引擎**: Unity 6 + URP
- **像素画**: Aseprite
- **3D 建模**: Blender
- **音频**: FMOD
- **版本控制**: Git + GitHub
